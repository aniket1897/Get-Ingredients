# Project
## Cloning
```
git clone git@github.com:aniket1897/Get-Recipe.git 
```

## Adding Database .sql file
1. Open ``` localhost/phpmyadmin  ```
1. Click on ```Imports```.
1. Click on ```choose file```.
1. And click ```Go```
